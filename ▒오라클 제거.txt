-- ○ 주요 버전 구분
-- Express Edition
-- : 법적으로 완전 무료 버전이다.
-- 기업체나 교육기관 등에서 무료 사용이 가능한 버전이며
-- 프로그램 개발(C#.NET, ASP.NET, JAVA, JSP 등) 용으로는 충분하지만
-- 데이터 베이스 서버용으로는 다소 부족한 기능을 사진 버전이라 할 수 있다.
-- Standard Edition, Standard Editrion One, Data Were House, Enterprise Edition
 	: 오라클 홈페이지 회원가입을 통해 다운로드는 가능하지만
	  기업체나 교육기관 등에서 사용하게되면 사용중 검열 시 정식
	  라이센스를 제시 할 수 있어야한다.
	  프로그램 개발용 뿐 아니라, 데이터 베이스 서버용으로도
	  충분한 기능을 가지고 있는 버전들이다.
-- Oracle Sever 용으로 가장 충분한 기능을 가지고 있는 버전은
-- Enterprise Edition 이다.

--네이버 블로그 참조

-- 1. 제어판 > 앱(프로그램 및 기능) → Oracle Database 11g Express Edition

-- 2. 실행창 호출 (윈도우키 + r) → 『service.msc』 입력

-- 3. 위 항목을 토대로 확인하면 
-- Oracle 로 시작하는 서비스가 여럿 확인된다,.
-- 즉, Orcle Server 는 서비스를 기반으로 동작한다는 것이다.
-- 위의 『1.』 에서처럼 오라클은 프로그램을 제거했다 하더라도
-- 이 오라클 서비스를 제거해 주어야한다.

-- 오라클이 정상적으로 제거되지 않을 경우....
-- 많은 문제를 발생시키게된다.

-- 오라클 서비스를 제거하는 방법 
-- 실행창 호출 (윈도우키 + r) → 『regedit』입력(조심조심)

-- HKEY LOCAL MACHINE > SOFTWARE >
-- 『ORACLE』 항목 삭제

--  HKEY LOCAL MACHINE > SYSTEM >CurrentControlSet >Services
-- 『ORACLE』 로 시작하는 모든 항목 삭제

-- HKEY LOCAL MACHINE > SYSTEM >ControlSet001 >Services
-- 『ORACLE』 로 시작하는 모든 항목 삭제

-- HKEY LOCAL MACHINE > SYSTEM >ControlSet002 >Services
-- 『ORACLE』 로 시작하는 모든 항목 삭제


-- 4. 재부팅 이후 탐색기에서 오라클 홈(오라홈) 과 관련된 모든 항목을
--    물리적으로 삭제한다.

-- 5. 또한, 데이터 파일 경로 및 설치 경로의 모든 디렉터리와 파일들을
